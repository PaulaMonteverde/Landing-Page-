DIFERENCIAL PARA A SUA EVOLUÇÃO

Contraste e sua importancia

O contraste de cores é um dos elementos mais importantes do design de interfaces, especialmente quando se trata de acessibilidade.

A boa utilização das cores pode facilitar a leitura e a compreensão da interface, enquanto o mau uso pode dificultar a compreensão e até mesmo impedir o acesso para pessoas com deficiência visual.

DIRETRIZES PARA A WEB

Uma web para todos

A Web Content Accessibility Guidelines (WCAG) é um conjunto de diretrizes estabelecidas pelo World Wide Web Consortium (W3C) para tornar o conteúdo da web mais acessível para pessoas com deficiência.

A versão mais recente, a WCAG 2.1, inclui uma série de critérios para garantir que as interfaces sejam acessíveis para pessoas com deficiência visual.
